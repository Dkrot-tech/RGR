Описание
Данный проект реализует шифр Атбаш на языке C++.
Шифр Атбаш - это простой метод шифрования, при котором каждая буква алфавита заменяется на противоположную, например: A ↔ Z, B ↔ Y, C ↔ X и т.д.
Шифрование и дешифрование выполняются одной и той же функцией.

Структура проекта
main.cpp - точка входа в программу
atbash.h, atbash.cpp - реализация шифра Атбаш
file_manager.h, file_manager.cpp - чтение и запись файлов
key_generator.h, key_generator.cpp - генератор ключа(для соответствия ТЗ)
error_codes.h - коды ошибок

Функционал
Чтение текста из файла;
Шифрование/дешифрование методом Атбаш;
Запись результата в файл;
Обработка ошибок.

Особенности
Шифр Атбаш не использует ключ, но для соответствия техническому заданию добавлен генератор ключа.
Шифрование и дешифрование идентичны.
