# 🔐 Encryption Algorithm RGR

Программа для шифрования и дешифрования текста и файлов с использованием алгоритма XOR.

---

## 📋 Описание

Консольное приложение для шифрования текста и файлов с поддержкой динамически подключаемых плагинов.

**Основные возможности:**
- Шифрование/дешифрование текста
- Шифрование/дешифрование файлов (любые форматы)
- Генерация ключей
- Динамическая загрузка плагинов (.so)

## 🛠️ Сборка и запуск

```bash
# 1. Перейти в папку проекта
cd ~/EncryptionAlgorithmRGR

# 2. Собрать плагин XOR
g++ -shared -fPIC -std=c++17 -o xor_plugin.so Plugins/XORCipher/XORCipher.cpp

# 3. Собрать основное приложение
g++ -std=c++17 -o encryption_rgr \
    UI/ConsoleUI.cpp \
    Core/CipherManager.cpp \
    Core/FileProcessor.cpp \
    -ldl

# 4. Создать папку для плагинов
mkdir -p plugins

# 5. Скопировать плагин
cp xor_plugin.so plugins/

# 6. Запустить
./encryption_rgr
