# Шифр Виженера

## Описание
Проект реализует шифр Виженера на языке C++ с поддержкой кириллицы и латиницы.

Шифр Виженера - это метод полиалфавитного шифрования, использующий ключевое слово для сдвига букв в тексте.

## Особенности
- ✅ Поддержка кириллицы (русский алфавит)
- ✅ Поддержка латиницы (английский алфавит)
- ✅ Сохранение регистра букв
- ✅ Неалфавитные символы не изменяются
- ✅ Работа с файлами (текстовыми и бинарными)
- ✅ Генерация ключей
- ✅ Система обработки ошибок

## Структура проекта
- `main.cpp` - точка входа в программу
- `vigenere.h/cpp` - реализация шифра Виженера
- `file_manager.h/cpp` - чтение и запись файлов
- `key_generator.h/cpp` - генерация ключей
- `error_handler.h/cpp` - обработка ошибок
- `error_codes.h` - коды ошибок
- `Makefile` - файл для сборки
